## âœ–ï¸ Multiply Tuple 

In Python, you can multiply a tuple by an integer using the `*` operator. This creates a new tuple where the original elements are repeated multiple times.

> Example

>```python
>fruits = ("apple", "banana")
>
>result = fruits * 3
>print(result)
>```

**Output**

>```
>('apple', 'banana', 'apple', 'banana', 'apple', 'banana')
>```
