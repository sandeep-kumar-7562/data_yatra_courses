# ğŸ”„ Updating Tuples in Python

Tuples are **immutable**, which means once a tuple is created, **its elements cannot be changed** directly.

> Example

>**Convert the tuple into a list to be able to change it**:
>
>
>```python
># Original tuple
>my_tuple = ("apple", "banana", "cherry")
>
># Convert to list
>temp_list = list(my_tuple)
>
># Modify the item
>temp_list[1] = "blueberry"
>
># Convert back to tuple
>my_tuple = tuple(temp_list)
>
>print(my_tuple)
>```

**Output**
>```
>('apple', 'blueberry', 'cherry')
>```

## âŒ Remove items

Tuples do not support direct deletion. You can convert to a list and use .`remove()`:

> Example

>```python
>my_tuple = ("a", "b", "c")
>
># Convert to list
>temp_list = list(my_tuple)
>
># Remove item
>temp_list.remove("b")
>
># Convert back to tuple
>my_tuple = tuple(temp_list)
>
>print(my_tuple)
>```

**Output**

>```
>('a', 'c')
>```

The del keyword can `delete` the tuple completely:

> Example

>```python
>my_tuple = ("a", "b", "c")
>
>del my_tuple
>
>#this will raise an error because the tuple no longer exists
>print(my_tuple)
>```
